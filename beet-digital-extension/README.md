# Extensión de Chatwoot: Beet Digital

Este proyecto contiene el código para una extensión personalizada de Chatwoot, diseñada para mejorar la comunicación por canales y añadir una capa de automatización inteligente.

## Alcance del Proyecto

El objetivo es construir una integración nativa dentro de un fork de Chatwoot que provea:

1.  **Capacidades Avanzadas de WhatsApp:** Soporte extendido para mensajes interactivos, WhatsApp Flows, mensajes de producto y multimedia.
2.  **Automatización Inteligente:** Un motor de automatización basado en el **Model Context Protocol (MCP)** para ejecutar lógica de negocio del lado del servidor basada en el contexto de las conversaciones.

La arquitectura sigue el modelo de la edición Enterprise de Chatwoot, donde este código actúa como una capa de superposición que extiende la funcionalidad base.

Para un desglose detallado de las fases y tareas, consulta el [Plan de Trabajo](./plan_de_trabajo.md).
